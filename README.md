## Title

Natural Language Processing App

## Description

In order to complete the "Natural Language Processing App" course, this is an asynchronous web app that uses Web API (https://learn.meaningcloud.com/) to analyze URLs

## Getting started

* Clone the repository by using git clone.
* Installing dependecies with npm i.
* Start App by using npm run start-dev - it builds assets and runs the app.
* Open in your browser the following path localhost:8081
* And Enjoy it.

## Features
* Scss - https://sass-lang.com/guide
* Fetch
* Local Server using Express
* Build tool: Webpack
* Service workers
